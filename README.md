# public-demos

A collection of public demos, notebooks, and experimental projects across multiple domains.

This repository serves as a demo showcase and a working monorepo for small standalone projects. Each folder represents one independent demo or experiment.

## Categories

### UI / Frontend
- `animated-gradient-text-starter` — animated gradient text starter demo built with Next.js, TypeScript, and Tailwind CSS.

### Quant / Trading
- `binance-multi-assets-singal-agent` — Python-based multi-asset Binance signal agent with backend, frontend, data, and source agent components.

### Forecasting / Modeling
- `bond-weighted-forecast` — forecasting-related demo or experiment.
